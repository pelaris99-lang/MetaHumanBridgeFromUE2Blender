MetaForge

A Blender-side bridge for MetaHuman DCC exports: import the character, keep scale sane, connect materials, build usable control handles, and expose body-proportion workflows without hiding the technical state.

Read the README Engineering Notes Public Release Boundary
DCC ExportPoint the add-on at a local MetaHuman DCC export instead of committing private sample assets.
Import + MaterialsImport body and face data, keep world scale stable, and connect texture maps where available.
Control RigBuild larger selectable Blender handles while preserving original deformation rigs.
Shape ToolsUse explicit body-proportion rebuilds, torso guides, mirroring, and bake/copy paths.

Install Package

Generate the Blender install zip locally:

powershell -ExecutionPolicy Bypass -File .\scripts\package_release.ps1 -Version 0.1.40

The zip should contain metahuman_blender_pipeline/ at the top level.

Validation

Use the background scripts in tools/analyze/ for regression checks. Visual inspection is deliberately separate from this release pass.
