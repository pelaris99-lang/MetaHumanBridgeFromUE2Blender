# MetaForge Docs

This folder contains the public and continuation documentation for the `metahuman_blender_pipeline` Blender add-on.

Recommended reading order:

1. `PUBLIC_RELEASE.md` for the source/public-asset boundary.
2. `CLOTHING_WORKFLOW.md` for the direct clothing weight and manual Weight Paint workflow.
3. `TECHNICAL_NOTES.md` for implementation notes and repair history.
4. `TEST_INDEX.md` for background validation scripts and result files.
5. `ENGINEERING.md` for broader project structure.

Current plugin version: `0.1.50`.

Local MetaHuman sample exports, GraceSwat project assets, screenshots, and `.blend` files are intentionally outside public git.
